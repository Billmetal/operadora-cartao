# Operadora de Cartões

Projeto realizado para desafio para conhecimento em Programação Orientada a Objetos (POO) em Java.

## implementação

Este programa simula a criação de uma operadora de cartão , um cartão , uma conta e clientes , executa 
operações básicas de saque e transferência e visualização de saldo em conta.

### tecnologia

Orientação a Objetos na linguagem Java .
